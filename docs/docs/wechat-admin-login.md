# 微信管理员登录

后端正式环境需要配置 `WECHAT_APP_ID` 和 `WECHAT_APP_SECRET`。AppSecret 只允许保存在服务端环境变量中，不能写入小程序或提交 Git。

目标微信先登录一次小程序，服务端用 `code2Session` 获取该 AppID 下的唯一 OpenID 并在 Supabase 创建用户。超级管理员随后在独立网页后台的“系统设置 → 微信管理员授权”中选择该用户并授予角色。授权后，该微信再次打开同一个小程序会自动换取独立管理员会话并进入管理工作台。

微信小程序不会向开发者提供用户的个人微信号，因此后台使用昵称、内部用户 ID 和脱敏 OpenID 识别用户。`ENABLE_DEV_AUTH=true` 仅用于非生产环境，生产环境自动禁止开发身份。
